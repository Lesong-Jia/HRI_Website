
### Install Node (if need)

```
brew install node
```

### Install or Update npm

```
npm install npm -g
```

### Set up vue

```
yarn add ant-design-vue
```
### Check if node and npm successfully installed
```
node -v
npm -v
```
### In the '\src' folder
### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```
